## ⚡ 快速开始

### 第一步：搭建学习环境

```bash
# 1. 安装 Miniconda（推荐）
# 下载地址：https://docs.conda.io/en/latest/miniconda.html

# 2. 创建 ML 专用环境
conda create -n ml python=3.11 -y
conda activate ml

# 3. 安装核心依赖
pip install numpy pandas matplotlib seaborn scipy          # 科学计算
pip install scikit-learn xgboost lightgbm optuna shap      # 机器学习
pip install torch torchvision --index-url https://download.pytorch.org/whl/cpu  # 深度学习(CPU)
pip install transformers datasets accelerate peft          # NLP/大模型
pip install gradio streamlit fastapi uvicorn               # 部署&Demo
pip install mlflow wandb dvc                               # MLOps

# 4. 验证安装
python -c "import torch; import sklearn; import transformers; print('✅ 环境配置成功！')"
```

### 第二步：克隆并开始学习

```bash
git clone https://github.com/Wxt19840820/ML-STUDY.git
cd ML-STUDY

# 从 Phase 1 数学基础开始
cat phase1-math/README.md
```

## 🎓 学习原则

```
✅ 每天写代码     哪怕只有 30 分钟，动手比看视频更重要
✅ 先直觉后公式   理解"为什么"比记住"是什么"更有价值
✅ 项目驱动       每阶段末完成一个真实可展示的项目
✅ 记录与复盘     每天填 daily-log.md，每周末总结一次
✅ 不怕出错       报错是最好的老师，Debug 是核心技能
```

## 💬 常见问题

<details>
<summary><b>Q: 没有数学基础可以学吗？</b></summary>
<br/>
完全可以。Phase 1 从最基础的概念讲起，推荐 3Blue1Brown 系列，用动画讲直觉，不需要死记公式。高中数学水平就够入门。
</details>

<details>
<summary><b>Q: 没有 GPU 怎么办？</b></summary>
<br/>
前三个阶段完全不需要 GPU。Phase 4 开始可以用 Google Colab（免费T4）或 Kaggle Notebooks（免费P100，每周30小时），完全够用。
</details>

<details>
<summary><b>Q: 每天只有 1 小时，可以学吗？</b></summary>
<br/>
可以，只是时间线需要延长到 18-20 个月左右。核心是"每天都动手"，比一周偷懒六天然后一天学 8 小时效果好很多。
</details>

<details>
<summary><b>Q: Phase 5 应该选哪个方向？</b></summary>
<br/>

- **NLP**：就业面最广，大模型时代最热门，推荐大多数人选这个
- **CV**：图像视频相关，落地场景直观（自动驾驶、安防、医疗影像）
- **RL**：门槛最高，适合对算法和游戏感兴趣的同学

</details>

<details>
<summary><b>Q: 学完能找到工作吗？</b></summary>
<br/>
完成 Phase 1-4 后可以竞争初级 ML 工程师岗位。Phase 5-6 完成后，拥有完整项目作品集，竞争力会大幅提升。关键是每个阶段都要做真实可展示的项目。
</details>
