## 🎯 学习原则

1. **每天写代码** — 哪怕只有 30 分钟，动手比看视频更重要
2. **先直觉后公式** — 理解"为什么"比记住"是什么"更有价值
3. **项目驱动** — 每阶段末尾完成一个真实可展示的项目
4. **记录与复盘** — 每天写 `daily-log.md`，每周末复盘一次
