# 🎯 Phase 5 · 方向专精

**时间：第 37–44 周（共 56 天）**
**目标：在一个方向建立作品集，能独立处理该领域真实问题**

> ⚠️ 只选一个方向深入，另外两个方向了解基本概念即可

## 三个方向

| 方向 | 适合人群 | 就业方向 |
|------|---------|---------|

| C：RL | 对游戏/机器人/决策感兴趣 | 算法研究员 |
